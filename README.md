# 🦷 Sistema de Gestión Odontológica

Sistema integral para clínicas dentales desarrollado con **Node.js**, **Express** y **SQLite**. Permite la gestión de pacientes, control de citas y seguimiento de historias clínicas (evoluciones).

## 🚀 Características

- **Gestión de Pacientes:** Registro, edición y eliminación de datos personales.
- **Agenda Médica:** Programación de citas con validación de pacientes.
- **Historia Clínica:** Registro cronológico de evoluciones por paciente.
- **Interfaz Moderna:** Diseño responsivo con Bootstrap 5 y animaciones suaves.

## 🛠️ Tecnologías utilizadas

- **Backend:** Node.js, Express.js
- **Base de Datos:** SQLite (vía Sequelize/sqlite3)
- **Frontend:** HTML5, CSS3 (Variables y Animaciones), JavaScript (Vanilla)
- **Estilos:** Bootstrap 5, Bootstrap Icons

## 📦 Instalación

1. Clonar el repositorio:
   `git clone https://github.com/TU_USUARIO/TU_REPOSITORIO.git`
2. Instalar dependencias:
   `npm install`
3. Iniciar el servidor:
   `npm start`
